Static Backdrop:
Static Backdrop Modal
When using the data-backdrop='static' attribute with a modal window, the user cannot dismiss the modal by clicking outside of the modal pane.
  • HTML
  • PUG 
//- Button trigger modal
button.btn.btn-primary(type='button', data-toggle='modal', data-target='#staticBackdrop')
    | Launch Static Backdrop Modal
//- Modal
#staticBackdrop.modal.fade(data-backdrop='static', tabindex='-1', role='dialog', aria-labelledby='staticBackdropLabel', aria-hidden='true')
    .modal-dialog(role='document')
        .modal-content
            .modal-header
                h5#staticBackdropLabel.modal-title Static Backdrop Modal
                button.close(type='button', data-dismiss='modal', aria-label='Close')
                    span(aria-hidden='true') ×
            .modal-body
                | ...
            .modal-footer
                button.btn.btn-secondary(type='button', data-dismiss='modal') Close
                button.btn.btn-primary(type='button') Understood
The static backdrop modal option cannot be dismissed by clicking outside of the modal pane. Instead, the user must dismiss the modal by using the close button within the modal pane.
